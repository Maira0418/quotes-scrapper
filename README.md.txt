# 📝 Quotes Scraper

This is a simple Python script that scrapes quotes, authors, and tags from [quotes.toscrape.com](http://quotes.toscrape.com) using `requests`, `BeautifulSoup`, and `pandas`.

## 🚀 Features
- Scrapes quote text, author, and tags
- Loops through multiple pages
- Saves data to CSV file

## 🛠️ Tools Used
- Python 3
- Requests
- BeautifulSoup
- Pandas

## 📦 Installation

```bash
pip install -r requirements.txt
