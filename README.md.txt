# 📝 Quotes Scraper

This is a beginner-friendly Python project that scrapes quotes, authors, and tags from [quotes.toscrape.com](http://quotes.toscrape.com) using `requests`, `BeautifulSoup`, and `pandas`.

Built by [Maira Qureshi](https://github.com/Maira0418), this scraper saves data into a clean CSV format that can be used for analysis, automation, or portfolio work.

---

## 🚀 Features

- 📄 Scrapes quote text, author names, and tags
- 🔁 Loops through multiple pages
- 💾 Saves results to `quotes.csv`
- 💡 Simple, readable code — great for beginners

---

## 🛠️ Tools & Libraries

- 🐍 Python 3.x
- 📦 `requests` – HTTP requests
- 🍜 `BeautifulSoup` – HTML parsing
- 📊 `pandas` – Data handling and CSV export
